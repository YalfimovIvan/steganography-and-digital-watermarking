# Стеганография и цифровые водяные знаки

Репозиторий объединяет три самостоятельных эксперимента из области сокрытия информации в изображениях:
- встраивание цифрового водяного знака в DCT-области
- QIM-стеганография в пространственной области
- стегоанализ изображений на основе признаков и классификатора KNN

## Содержимое

### `dct_watermarking.py`

Встраивание и извлечения цифрового водяного знака.

Как работает:
- изображение приводится к размеру `512x512`
- переход выполняется в пространство `YCrCb`
- водяной знак встраивается по блокам `8x8`
- в каждом блоке считается DCT
- коэффициент `(3, 3)` модифицируется с силой `alpha`
- затем выполняется обратное DCT

Что ещё умеет:
- считать `PSNR` между исходным и изменённым изображением
- считать `NCC` между исходным и извлечённым водяным знаком
- проводить эксперимент на робастность
- тестировать устойчивость к JPEG-сжатию, шуму и изменению яркости

### `qim_steganography.py`

Сокрытие текстового сообщения в изображении методом Quantization Index Modulation.

Как работает:
- текст переводится в бинарную строку
- в начало записывается 32-битный заголовок с длиной сообщения
- каждый бит кодируется изменением значения пикселя относительно шага квантования `q`
- для извлечения проверяется, к какому из двух квантованных состояний ближе пиксель

Что ещё умеет:
- считать `PSNR`
- строить гистограммы каналов
- проводить эксперимент на максимальную загрузку
- сравнивать качество при разной степени загрузки
- оценивать `BER` после атак: JPEG, шум, изменение яркости, масштабирование

### `steganalysis_classifier.py`

Построение простого стегоанализатора на изображениях.

Конвейер работы:
1. Создание стегоизображений через `stegano.lsb`.
2. Проверка качества встраивания.
3. Извлечение признаков Фридриха из DCT-представления.
4. Добавление статистических, текстурных и спектральных признаков.
5. Нормализация признаков.
6. Обучение `KNeighborsClassifier`.
7. Оценка качества и запуск экспериментов по разным `k` и метрикам расстояния.

Какие признаки используются:
- модифицированные признаки Фридриха
- гистограммы DCT-коэффициентов
- среднее, стандартное отклонение, медиана
- характеристики спектра Фурье
- энтропия
- признаки на основе операторов Собеля
